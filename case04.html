Project Overview

A scalable system for promotional banner creation.

Car dealers required monthly promotional banners aligned with seasonal events and holidays. The existing process was manual, time-consuming, and lacked clarity for both dealers and internal teams.


The opportunity: create a self-serve system that simplifies banner selection and streamlines communication.

My Role Product Designer
Platform Web-based Tool
Timeline ~4 Weeks
Outcome +60% faster banner selection time
The Problem

Manual process. Limited clarity.

Dealers relied on back-and-forth communication to choose banners, making the process inefficient and difficult to scale.

Signal 01
High Coordination Effort
Teams had to individually share multiple banner options with each dealer.
Signal 02
Decision Friction
Dealers struggled to communicate preferences clearly, delaying final selections.
Growth Strategy

Self-serve customization with clarity.

Pillar 01

Reduce communication overhead

Centralized all banner options into a single interactive experience, eliminating repetitive manual sharing.

Pillar 02

Enable visual decision-making

Allowed users to mix and match elements with real-time preview, making selection intuitive and faster.

Solution

Self-Serve Banner Builder

A simple, interactive web tool that allows dealers to create and customize promotional banners by selecting predefined elements and generating a shareable code.

Solution 01
Step-by-step banner creation seamless flow
Solution 02
Auto-generated export code for easy sharing
Solution 03
Mix-and-match title, background, and offer elements
Solution 04
Live preview of banner combinations with centralized access
Impact

Faster workflows.
Better collaboration.
